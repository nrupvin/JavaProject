## ‚ú® Key Takeaways

### The Big Picture
- **Maven** = Build tool (compiles code, downloads libraries)
- **Spring Boot** = Framework (provides web server, templates)
- **Java 17** = Programming language (you write code in this)
- **pom.xml** = Project blueprint (tells Maven what to do)
- **Application** = Runs on port 8080, handles HTTP requests

### The Process
```
Write Code ‚Üí Build (mvn) ‚Üí Test ‚Üí Run ‚Üí Access Browser
```

### The Commands You'll Use Most
```powershell
mvn clean package          # Build
mvn spring-boot:run        # Run
mvn test                   # Test
```
